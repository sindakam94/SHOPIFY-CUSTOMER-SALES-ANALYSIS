🛍️ Shopify Sales and Customer Funnel Report Dashboard

An interactive and analytical Microsoft Power BI dashboard designed to analyze Shopify sales performance, customer purchasing behavior, retention trends, product performance, payment preferences, and regional sales distribution across the United States.

📌 Short Description / Purpose

The Shopify Sales and Customer Funnel Report is a dynamic Power BI dashboard built to help businesses monitor e-commerce performance through key operational and customer-focused metrics.

The dashboard provides actionable insights into:

Net sales performance
Customer retention behavior
Product category contribution
Payment method trends
Geographic sales concentration
Order quantity analysis
Customer funnel metrics

This dashboard is intended for:

E-commerce analysts
Shopify store owners
Sales managers
Marketing teams
Business intelligence professionals
Data analytics portfolio projects
🛠️ Tech Stack

The dashboard was built using the following tools and technologies:

📊 Microsoft Power BI – Main data visualization platform used for dashboard development.
📂 Power Query – Used for data cleaning, transformation, and preparation.
🧠 DAX (Data Analysis Expressions) – Used for calculated KPIs, measures, aggregations, and dynamic filtering.
🗂️ Data Modeling – Relationships established between sales, customers, products, geography, and payment datasets.
🌍 Bing Maps Integration – Used for geographical and regional sales visualization.

✨ Features / Highlights
📉 Business Problem

E-commerce businesses generate large amounts of transactional and behavioral data, but decision-makers often struggle to quickly identify:

Which products generate the most revenue?
Which customer segments are most valuable?
What payment methods dominate purchases?
Which regions drive the highest sales?
How effective is customer retention?
What purchasing patterns occur throughout the day?

Without centralized analytics, identifying growth opportunities and operational bottlenecks becomes difficult.

🎯 Goal of the Dashboard

To create an interactive analytics solution that:

Tracks overall sales performance
Monitors customer retention and repeat behavior
Identifies top-performing product categories
Analyzes customer and order trends by hour
Evaluates payment method preferences
Visualizes regional sales performance
Supports strategic e-commerce decision-making

Repeat customer rate of 46.02% indicates strong customer loyalty.
